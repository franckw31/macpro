📱 Écriture NFC (Web NFC)

ℹ️ Compatibilité : Cette fonctionnalité utilise la Web NFC API et est disponible sur :
  • Chrome sur Android 6.0+
  • Fonctionne sur localhost sans HTTPS
  • HTTPS requis uniquement en production (domaine public)
  • Permissions NFC activées

📋 Instructions :
  1. Entrez le texte à écrire ci-dessus
  2. Cliquez sur le bouton "Écrire sur la puce NFC"
  3. Maintenez l'arrière de votre appareil contre la puce NFC
  4. Attendez la confirmation de l'écriture
  5. Un message de succès s'affichera
