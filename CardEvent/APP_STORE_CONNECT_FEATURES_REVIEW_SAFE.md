# CardEvent — App Review Safe Description

Use this version if Apple asks for a more neutral and compliance-focused explanation of the app.

## Detailed information on app features

CardEvent is a private member companion app used to manage and follow live card events organized by a closed community.

The app provides the following features:

- Secure member login
- Viewing upcoming and current event information
- Managing event participation and registration status
- Viewing tournament structure and timer information
- Following a live event timer with blinds and level progression
- Viewing participant and player profile information related to the event
- Viewing challenge rankings and event statistics
- Viewing ticket history and member-related event tracking data
- Viewing payout or prize distribution information for event results
- Receiving push notifications for event-related updates when enabled

## App Review clarification

- The app is intended for members of a private card event community.
- The app does not offer gambling gameplay, casino games, sports betting, or real-money wagering.
- It does not process bets, deposits, withdrawals, or financial transactions.
- Any displayed monetary values are informational only and relate to live event organization, such as entry fee, tournament structure, or published event results.
- The app is used for event coordination, live information display, and member access to event-related data.
