# CardEvent — App Features Description for App Store Connect

Below is a ready-to-submit description you can use to answer Apple's request for detailed information on what the app features are.

## Detailed information on app features

CardEvent is a companion app for live poker tournament and card event management. It is designed for registered members of a private card community and provides tools to view event information, follow tournament progress, manage participation, and access personal event-related data.

The app includes the following core features:

1. **User authentication**
   - Members can securely sign in to access their account-specific information.
   - The app stores the authentication token securely in the device keychain for future sessions.

2. **Upcoming and current event overview**
   - Users can view the next scheduled card event or tournament.
   - The home screen displays key information such as event title, date, organizer, entry fee, rake, recave information, and registration status.

3. **Tournament registration and participation status**
   - Users can register for an event, update their participation status, or view their current registration state.
   - Optional flags such as anonymous participation, late registration, or option/waitlist status can be managed depending on event settings.

4. **Live tournament timer**
   - The app includes a local blind timer for tournament use.
   - It displays the current blind level, countdown, next blind level, and tournament structure information.
   - The timer can be started, paused, reset, and adjusted by the user.

5. **Live event timer view**
   - Users can open a live view for the current event to follow tournament progression in real time.
   - This includes remaining time, current blinds, ante, pause state, player counts, and average stack when provided by the server.

6. **Player and participant information**
   - Users can view the participant list for an event.
   - The app provides individual player profile information related to the current event, including registration data, bonus information, recaves, bounty, ranking, gains, and challenge rank when available.

7. **Challenge ranking view**
   - Users can view challenge rankings associated with tournament activity.
   - Rankings can include participant position, points, final table appearances, victories, and participation count.
   - Users can switch between available challenges when the server provides multiple ranking groups.

8. **Prize pool / payout calculation**
   - The app includes a payout and prizepool calculation section.
   - Users can review or calculate distribution-related information for tournament results.

9. **Tickets and member-related event rewards**
   - Users can view their event-related ticket history and monthly totals inside the app.
   - This feature is intended for community/member tracking and visibility.

10. **In-app help and utility views**
   - The app contains supporting views such as help, activity details, tournament structure, and profile-related navigation to assist users during live events.

11. **Push notification support**
   - The app can register the device for remote notifications when enabled by the user and configured by the operator.
   - Notifications are used to inform members about event-related updates.

## Additional clarification for App Review

- CardEvent is not a gambling app and does not process real-money wagering, betting, or casino gameplay.
- The app is used as an event companion and tournament information tool for a live poker community.
- Any monetary values shown in the app correspond to event organization data such as entry fee, rake, recaves, payout calculation, or winnings/results display for community events.
- The app does not sell digital content, does not provide in-app purchases, and does not unlock paid app functionality.

## Short version (if Apple requests a shorter answer)

CardEvent is a companion app for a private live poker event community. It allows authenticated members to view upcoming events, register for tournaments, follow a live tournament timer, view participant and player information, check challenge rankings, access ticket history, and review prizepool or payout-related data. The app is informational and organizational in nature and does not provide gambling gameplay or betting features.
